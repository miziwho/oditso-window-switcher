창전환 (Oditso - Window Switcher)

Alt+Tab보다 빠르고 안정적인 Windows 창 관리 도구

여러 창을 오가느라 작업 흐름이 끊기고 불편했던 경험이 있다면,
창전환(Oditso)은 당신의 생산성을 극대화해주는 새로운 방식의 창 전환 솔루션입니다.




📖 소개

**창전환(Oditso - Window Switcher)**는 복잡한 창 전환을 더 빠르고 정확하게 처리하기 위해 개발된
Windows용 창 관리 도구입니다.

시각적 창 목록, 정렬, 색상 강조, Bold 표시 등
기존 Alt+Tab에서 제공하지 않는 기능들을 통해 원하는 창을 즉시, 정확하게 찾아갈 수 있습니다.

대기업 환경에서도 검증된 안정성과 속도를 기반으로,
수십 개 창을 동시에 사용하는 실무자에게 특히 최적화되어 있습니다.

✨ 주요 기능
🎯 핵심 기능

창 목록 시각화 — 열린 창을 리스트 형태로 정리하여 표시

순서 재배치 — ⏫ 맨위 / ▲ 위로 / ▼ 아래 / ⏬ 맨아래 이동

굵게(Bold) 강조 — 중요한 창을 한눈에 표시

행별 색상 설정

글꼴색: 빨강, 주황, 초록, 파랑, 갈색, 검정

배경색: 노랑, 회색, 흰색

멀티 선택 — Ctrl+클릭 또는 Shift+클릭으로 여러 창 제어

창 이름 변경 — 원하는 창에 별도 제목 설정 가능

창 상태 표시 — 최소화 / 최대화 / 일반 상태 실시간 표시

🎨 커스터마이징

글꼴 3종 제공: 굴림 / 돋움 / 맑은 고딕

글자 크기 4단계 지원: 8 / 10 / 12 / 14pt

다국어 지원: 한국어·English 자동 저장

F5 새로고침 — 창 목록 즉시 업데이트
